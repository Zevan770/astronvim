# Zev's AstroNvim config

**NOTE:** AstroNvim v5+

## 🛠️ Installation

### Clone the repository

```shell {#clone}
git clone https://github.com/zevan770/astronvim ~/.config/astronvim
```

### Start Neovim

```shell {#start}
NVIM_APPNAME=astronvim nvim
```
